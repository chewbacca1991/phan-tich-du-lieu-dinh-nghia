# Phân Tích Dữ Liệu Định Nghĩa

## Giới thiệu
Dự án này cho phép người dùng nhập dữ liệu định nghĩa từ nhiều nguồn và thực hiện phân tích trên dữ liệu đó. Bạn có thể tìm thấy thông tin về kinh tế, xã hội, môi trường và hơn thế nữa. Hệ thống sẽ hỗ trợ việc tạo ra các biểu đồ và báo cáo trực quan dễ hiểu.

## Tính năng
- Tải lên dữ liệu từ file CSV hoặc Excel
- Phân tích và trực quan hóa dữ liệu
- Chia sẻ báo cáo và biểu đồ với cộng đồng
- Hỗ trợ đa ngôn ngữ, bao gồm cả tiếng Việt

## Cài đặt
1. Clone repository này về máy của bạn.
2. Cài đặt các dependencies cần thiết.
3. Chạy ứng dụng và bắt đầu phân tích dữ liệu của bạn ngay!
